# JavaScript basics

## Lab environment

To launch the notebook server for this lab:
### Bash
```
docker run --name js-basics --rm -v "$(pwd)"/lab:/usr/project/lab -p 8888:8888 adrianbranescu93/jupyter-node
```

### PowerShell
```
docker run --name js-basics --rm -v "$pwd\lab:/usr/project/lab" -p 8888:8888 adrianbranescu93/jupyter-node
```
